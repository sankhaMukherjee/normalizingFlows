Single flow¶

In this first example, consider that you want to transform a latent variable \(\mathbf y\) which has a PDF \(q_{\mathbf y}(\mathbf y)\) to another latent variable \(\mathbf z\) that has a probability PDF \(q_{\mathbf z}(\mathbf z)\) closer to the density that we desire, which happens to be \(p_{\mathbf y}(\mathbf y)\). For this, we should be able to create a function mapping \(f_{\mathbf\theta}\) parametrized by \(\mathbf \theta\) such that

  1. \(\mathbf z = f_{\mathbf \theta}(\mathbf y)\)

  2. \(\mathbf y = f_{\mathbf \theta}^{-1}(\mathbf z)\)

We know that the KL divergence between our desired distribution and the new distribution is \(D_{KL}( p_{\mathbf y}(\mathbf y) || q_{\mathbf z}(\mathbf z) )\)

Note that we can, due to the change of variable formula, represent \(q_{\mathbf z}(\mathbf z)\) in terms of \(q_{\mathbf y}(\mathbf y)\) as:

\[q_{\mathbf z}(\mathbf z) = q_{\mathbf y}(\mathbf y) \left| \det \mathbf \nabla_{\mathbf y} f_{\theta}(\mathbf y) \right|^{-1}\]

This means that we should be able to find a set of parameters \(\theta\) such that, minimizing the parameters will allow us to find the best-case functional mapping for this new latent space. If we substitute this form in the KL divergence, we get the formula

\[D_{KL}( p_{\mathbf y}(\mathbf y) || q_{\mathbf z}(\mathbf z) ) = D_{KL} \left( p_{\mathbf y}(\mathbf y) || q_{\mathbf y}(\mathbf y) \left| \det \mathbf \nabla_{\mathbf y} f_{\theta}(\mathbf y) \right|^{-1} \right)\]
